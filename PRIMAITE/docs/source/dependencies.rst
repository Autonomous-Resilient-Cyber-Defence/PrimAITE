PrimAITE Dependencies
=====================

PrimAITE is built with the following versions of dependencies:

* Python 3.10.9
* PyYAML 6.0
* numpy 1.23.5
* networkx 2.8.8
* gym 0.21.0
* matplotlib 3.6.2 
* stable_baselines_3 1.6.2

The latest release of PrimAITE has been tested against the following versions of dependencies:

* Python 3.10.9
* PyYAML 6.0
* numpy 1.23.5
* networkx 2.8.8
* gym 0.21.0
* matplotlib 3.6.2 
* stable_baselines_3 1.6.2
